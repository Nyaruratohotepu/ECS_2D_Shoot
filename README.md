# ECS_2D_Shoot

本项目是一个基于ECS架构和传统MonoBehaviour的2D射击小游戏。
本项目为中国科学技术大学软件学院2018届《基于ECS架构的Unity游戏开发》课程实践项目开发。
本项目仅用于学习交流使用，严禁用于各种商业用途。
素材（包括图像、声音、字体等）均来自于互联网，如有侵权请联系本人删除。

unity 2018.3.14f1 Personal
